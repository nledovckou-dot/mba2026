# Быстрый деплой презентации

## Способ 1: Netlify Drop (Самый простой - 30 секунд)

1. Откройте: https://app.netlify.com/drop
2. Перетащите файл `Анализ_рынка_презентация.html`
3. Получите ссылку и поделитесь!

---

## Способ 2: Через Netlify CLI (Автоматический)

Если хотите автоматизировать:

```bash
# Установите Netlify CLI
npm install -g netlify-cli

# Войдите в аккаунт (откроется браузер)
netlify login

# Задеплойте файл
netlify deploy --prod --dir=. --file=Анализ_рынка_презентация.html
```

---

## Способ 3: GitHub Pages

1. Создайте репозиторий на GitHub
2. Загрузите файл и переименуйте в `index.html`
3. Включите GitHub Pages в настройках
4. Получите ссылку: `https://ваш-username.github.io/репозиторий`

---

## Способ 4: Просто отправить файл

Отправьте `Анализ_рынка_презентация.html` через:
- Email
- Telegram/WhatsApp  
- Google Drive / Яндекс.Диск

Файл откроется в браузере и будет работать!
