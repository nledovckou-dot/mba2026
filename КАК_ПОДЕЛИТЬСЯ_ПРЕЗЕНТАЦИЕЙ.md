# Как поделиться презентацией

## Вариант 1: Netlify Drop (Самый простой, без регистрации)

1. Откройте сайт: https://app.netlify.com/drop
2. Перетащите файл `Анализ_рынка_презентация.html` в окно браузера
3. Получите ссылку вида: `https://random-name-123.netlify.app`
4. Поделитесь ссылкой с коллегами

**Преимущества:**
- Не требует регистрации
- Работает сразу
- Бесплатно
- Ссылка действует пока файл не удалят

---

## Вариант 2: GitHub Pages (Постоянная ссылка)

Если у вас есть аккаунт GitHub:

1. Создайте новый репозиторий на GitHub
2. Загрузите файл `Анализ_рынка_презентация.html`
3. Переименуйте его в `index.html`
4. Включите GitHub Pages в настройках репозитория
5. Получите ссылку: `https://ваш-username.github.io/название-репозитория`

---

## Вариант 3: Отправить файл напрямую

Просто отправьте файл `Анализ_рынка_презентация.html` через:
- Email
- Telegram / WhatsApp
- Google Drive / Яндекс.Диск
- Любой мессенджер

Получатель откроет файл в браузере - все будет работать!

---

## Вариант 4: PDF (для печати/презентации)

Можно экспортировать в PDF через браузер:
1. Откройте `Анализ_рынка_презентация.html` в браузере
2. Нажмите Ctrl+P (или Cmd+P на Mac)
3. Выберите "Сохранить как PDF"
4. Сохраните файл

**Примечание:** В PDF не будет интерактивности (переключение слайдов), но можно распечатать каждый слайд отдельно.

---

## Рекомендация

Для быстрого доступа используйте **Netlify Drop** - это самый простой способ получить публичную ссылку за 30 секунд.
